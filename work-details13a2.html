In this assignment, I had to design, create, test and evaluate a website for an upcoming festival.
The requirements for the website are as follows:
• A website containing at least 8 linked interactive webpages.
o An attention-grabbing index page (home page).
o A line up page including images of the bands that are playing.
o Headliner page displaying information about the headlining band including interactive video media of the band, history and favourite songs.
o Newsletter subscription page.
o Location and directions page (using Google Maps embedded link).
o E-commerce page for merchandise, showing the merchandise in a table with prices.
o E-Commerce page for tickets sales.
o Contact us page (Email links, address, phone number, social media links).
• The focus of the website is to promote, advertise and sell tickets for an upcoming music festival.
• The website must load quickly (performant and optimised) and be simple to navigate.
• The website must contain a table formatted correctly to display the bands that are performing in a clear manner.
• The site must display a consistent use of text, fonts, colour schemes and page styles in all pages.
• The website must contain a Newsletter sign up form.
• The website must make use of digital media. (e.g. Images, Video, Animation)
• The website must have at least two external hyperlinks to other interesting webpages.
